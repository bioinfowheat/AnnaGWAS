1. The Core Problem: Reference Bias

When performing introgression analysis on backcross or admixed populations, the choice of reference genome has profound consequences for every downstream step. If the reference is unrelated to (or substantially divergent from) the individuals in the crosses, three compounding biases emerge:

Three cascading biases

  1. Mapping bias — reads carrying non-reference alleles map less efficiently
  2. Variant calling bias — REF/ALT labels become arbitrary with respect to parental lineages
  3. Haplotype inference bias — ancestry tract detection is compromised

Each bias feeds into the next, amplifying the distortion at every analytical layer.

2. Mapping Bias

Short reads containing alleles that differ from the reference are penalised during alignment. This creates asymmetric allele dropout: the parental lineage more distant from the reference loses reads at heterozygous sites, while the lineage closer to the reference is overrepresented.

Figure 1. Reads from the divergent parental lineage (Parent B, red) experience lower mapping quality and higher dropout rates at variant sites compared to the lineage closer to the reference (Parent A, green). At heterozygous sites, this asymmetry can cause the genotype caller to produce false homozygous calls for the reference-proximal allele.

Günther & Nettelblad (2019) showed this effect is pervasive in ancient DNA datasets and scales with fragment length. The effect is exacerbated at low coverage — exactly the conditions common in population resequencing studies. More recently, Duntsch et al. (2025) demonstrated in gray foxes that mapping to a heterospecific reference loses 26–32% of SNPs compared to a conspecific reference, and dramatically alters FST, nucleotide diversity, and effective population size estimates.

3. Variant Calling Bias

Even when reads from both parental lineages map successfully, the REF/ALT labels assigned during variant calling are arbitrary with respect to parental ancestry when using an unrelated reference. This creates a fundamental problem for introgression analysis.

Figure 2. When one parent is used as the reference (left), REF consistently represents Parent A alleles and ALT represents Parent B alleles. The genotype encoding (0/0, 0/1, 1/1) directly maps to parental ancestry. With an unrelated reference (right), the REF allele may happen to match Parent A at some sites and Parent B at others, or may carry a third allele (X) at some sites entirely. The 0/1 genotype encoding loses its ancestry meaning.

4. Haplotype Inference Problems

For introgression analysis, the goal is to identify contiguous blocks of ancestry from each parental lineage along each chromosome. This requires:

  • Identifying diagnostic SNPs (sites fixed or nearly fixed for different alleles in each parent)
  • Phasing alleles into haplotypes along chromosomes
  • Applying ancestry HMMs or similar methods to infer ancestry tracts

An unrelated reference degrades all three steps.

Figure 3. Panel A: When Parent A is used as the reference, all diagnostic SNPs are retained, REF (0) consistently means "Parent A allele", and ancestry tracts are cleanly delimited. A single recombination breakpoint is correctly detected. Panel B: With an unrelated reference, some diagnostic SNPs are lost to mapping failure (✗), and among the retained sites, 0 (REF) sometimes corresponds to Parent A and sometimes to Parent B (depending on whether the unrelated reference happens to share an allele with one parent). The HMM sees a noisy, fragmented signal and infers multiple spurious recombination breakpoints.

5. Solutions & Mitigations

Ranked by effectiveness

Approach Description When to use
1. Use a parental genome as reference Map all reads to one parent's genome. REF = Parent A, ALT = Parent B. Ancestry assignment is straightforward. Best option if you have a genome assembly for one parent.
2. Build a pseudo-reference If you have WGS data from both parents, build a consensus assembly for one parent and use that. When parental WGS data exists but no assembly.
3. Dual-reference mapping Map reads to both parental references independently, then compare. Tools like EAGLE or custom pipelines can leverage this. When both parental assemblies exist.
4. Post-hoc repolarization Map to unrelated reference, call variants, then use sequenced parental samples to reclassify alleles as "Parent A" vs "Parent B" origin. When you have parental samples sequenced alongside backcross progeny. Some sensitivity lost from mapping bias.
5. Ancestry HMMs on read counts Tools like Ancestry_HMM (Corbett-Detig & Nielsen 2017) work directly on pileup data (raw allele counts) rather than called genotypes, partially mitigating reference bias at the genotype-calling step. For any admixed/introgressed populations. Particularly valuable at low coverage.
6. Pangenome / graph-based reference Use variation graph aligners (vg) or pangenomic references (PanVC 3) that incorporate population-level variation, reducing the single-reference bottleneck. When population-level variation data is available for the species.

6. Case Study: Paris et al. (2022)

Paper: "A large and diverse autosomal haplotype is associated with sex-linked colour polymorphism in the guppy"

Paris JR, Whiting JR, Daniel MJ, et al. (2022) Nature Communications 13: 1233

This paper examines colour polymorphism in Trinidadian guppies (Poecilia reticulata) using Pool-seq of four Iso-Y breeding lines and whole-genome sequencing of wild-caught individuals. While the paper does not directly address reference genome bias in introgression, it provides an instructive example of how allele frequency polarization and haplotype phasing depend on reference choice.

Relevance to the reference polarization problem

  1. Explicit allele frequency polarization. Paris et al. polarized allele frequencies to the major allele of Iso-Y9. This is a deliberate choice: different polarization choices would produce different allele frequency patterns and different haplotype interpretations. In an introgression context, polarizing to the wrong reference can mask or invert the signal of introgressed blocks.
  2. Complex haplotype structure. The authors found multiple derived haplotypes nested within an older haplotype on LG1. Detecting these haplotypes required careful analysis of bimodal allele frequency distributions. With an unrelated reference, the REF/ALT encoding at each site would be arbitrary, making these bimodal patterns much harder to detect and interpret.
  3. Genotype-dependent ancestry inference. The paper's haplotype analysis (Figure 5 in the paper) polarized phased haplotypes to the Iso-Y9 genotype, classifying each position as matching the "Iso-Y9 haplotype" or the "alternative haplotype." Breakpoints between phases were then identified. This approach is directly analogous to ancestry tract inference in introgression analysis — and it only works cleanly because the data was polarized to a meaningful reference.
  4. HOM REF / HOM ALT interpretation. The genotype plot in the paper (Figure 5a) shows long stretches of HOM REF (yellow) vs HOM ALT (purple) genotypes. These visual patterns, which reveal the haplotype structure, are entirely dependent on the reference allele definition. With an unrelated reference, the colour blocks would be scrambled and the haplotype structure would be obscured.

Take-home message from Paris et al.

Although the Paris et al. study used a conspecific guppy reference genome (appropriate for their within-species analysis), their analytical approach — particularly the allele frequency polarization to Iso-Y9 and the haplotype phasing against a reference genotype — illustrates exactly why reference choice matters. Had they used a distantly related fish genome as reference, the bimodal allele frequency patterns on LG1, the clean haplotype blocks, and the breakpoint detection would all have been compromised.

For your backcross introgression analysis in Pararge aegeria, the same principles apply with even greater urgency: if the reference genome is from a different subspecies or population, allele frequency patterns and haplotype blocks will be distorted at the very sites that carry the introgression signal.

7. Key Literature

Reference bias — foundational studies

  1. Günther T & Nettelblad C (2019). The presence and impact of reference bias on population genomic studies of prehistoric human populations. PLoS Genetics 15(7): e1008302. doi:10.1371/journal.pgen.1008302
    Key finding: Reference bias causes heterozygous sites to be falsely called as homozygous; effect scales inversely with fragment length.
  2. Peona V, Blom MPK, Xu L, et al. (2021). One is not enough: On the effects of reference genome for the mapping and subsequent analyses of short-reads. PLoS Computational Biology 17(1): e1008678. doi:10.1371/journal.pcbi.1008678
    Key finding: Even 2% reference divergence at low coverage shifts estimates of theta by 0.14–0.18-fold and distorts Tajima's D and Fay & Wu's H.
  3. Duntsch L, Lichtenauer S, Armstrong EE, et al. (2025). Reference genome choice compromises population genetic analyses. Cell. doi:10.1016/j.cell.2025.02.008
    Key finding: Conspecific mapping detects 26–32% more SNPs; FST, Ne, and selection scans all change substantially with heterospecific reference.
  4. Lin MJ, Iyer S, Chen NC & Langmead B (2024). Measuring, visualizing, and diagnosing reference bias with biastools. Genome Biology 25: 101. doi:10.1186/s13059-024-03240-8

Reference bias in introgression & comparative genomics

  1. Feulner PGD & De-Kayne R (2025). Reference genome bias in light of species-specific chromosomal reorganization and translocations. Genome Biology. doi:10.1186/s13059-025-03761-w
    Key finding: Distantly related reference leads to inaccurate detection of chromosomal inversions (wrong size and location).
  2. Bohling JH (2020). Evaluating the effect of reference genome divergence on the analysis of empirical RADseq datasets. Ecology and Evolution 10: 7585–7601. doi:10.1002/ece3.6483
    Key finding: Increasing divergence from reference reduces aligned reads, SNP counts, and FST estimates (populations appear less differentiated).
  3. Edelman NB, Frandsen PB, Miyagi M, et al. (2019). Genomic architecture and introgression shape a butterfly radiation. Science 366(6465): 594–599. doi:10.1126/science.aaw2090
    Key finding: Used clade-specific reference genomes for Heliconius, acknowledging that cross-clade mapping introduces bias.
  4. Zhang W, Dasmahapatra KK, Mallet J, et al. (2016). Genome-wide introgression among distantly related Heliconius butterfly species. Genome Biology 17: 25. doi:10.1186/s13059-016-0889-0

Ancestry HMM methods

  1. Corbett-Detig RB & Nielsen R (2017). A Hidden Markov Model approach for simultaneously estimating local ancestry and admixture time using next generation sequence data. PLoS Genetics 13(1): e1006529. doi:10.1371/journal.pgen.1006529
    Key finding: Ancestry_HMM works on raw read counts (pileup data) rather than called genotypes, partially mitigating reference bias.
  2. Dias-Alves T, Mairal J & Blum MGB (2018). Loter: A software package to infer local ancestry for a wide range of species. Molecular Biology and Evolution 35(9): 2318–2326. doi:10.1093/molbev/msy126

8. Practical Recommendations for Your P. aegeria Pipeline

What to do

  1. Minimum: Include parental samples in your sequencing batch. After variant calling, use diagnostic SNPs (fixed differences between parents) to reclassify each VCF site by parental origin.
  2. Preferred: If parental WGS data is available, build a consensus assembly for one parent (e.g., using bcftools consensus) and re-map all reads to that pseudo-reference.
  3. For ancestry tract inference: Use Ancestry_HMM with pileup-mode input. It handles reference bias better than genotype-based methods (Loter, ELAI) because it works directly on raw read counts.
  4. Quantify the bias: Use biastools (Lin et al. 2024) to measure reference bias in your dataset before and after any mitigation.
  5. If stuck with the current reference: At minimum, filter for diagnostic SNPs where both parents are sequenced and allele assignments are unambiguous, then repolarize your VCF.

What to avoid

  • Do not assume REF/ALT labels in your VCF correspond to parental ancestry.
  • Do not use genotype-based ancestry tools (Loter, LAMP) without first repolarizing your VCF to a meaningful reference.
  • Do not ignore the problem — even modest divergence (2–5%) between reference and study species can shift population genetic summary statistics by 15–30% (Peona et al. 2021).
