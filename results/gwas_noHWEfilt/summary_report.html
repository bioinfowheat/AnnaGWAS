Rationale

The focal locus controlling diapause (DD) vs non-diapause (NN) is expected to be homozygous reference in one group and homozygous alternate in the other, with few or no heterozygotes. Such a locus will strongly violate Hardy-Weinberg equilibrium and would be removed by a standard HWE filter (p < 1×10−6). This analysis repeats the GWAS without the HWE filter to retain these variants.

Step 1 — Variant QC (no HWE filter)
plink2 --bfile results/plink/variants \
    --maf 0.05 \
    --geno 0.1 \
    --make-bed \
    --out results/plink/variants_qc_noHWE \
    --chr-set 27 --allow-extra-chr
Step 2 — Sample QC (missingness filter)
plink2 --bfile results/plink/variants_qc_noHWE \
    --mind 0.1 \
    --make-bed \
    --out results/plink/variants_qc_samples_noHWE \
    --chr-set 27 --allow-extra-chr
Step 3 — GWAS association (logistic regression, Firth fallback)
plink2 --bfile results/plink/variants_qc_samples_noHWE \
    --pheno results/gwas_noHWEfilt/phenotypes.txt \
    --covar results/pca/pca.eigenvec \
    --covar-col-nums 3-12 \
    --glm firth-fallback hide-covar \
    --out results/gwas_noHWEfilt/association_results \
    --chr-set 27 --allow-extra-chr

PCA covariates (10 PCs) are from the HWE-filtered dataset — standard practice for population structure estimation.
